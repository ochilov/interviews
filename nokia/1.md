# Поиск простых чисел

Простое число - согласно [wiki](https://ru.wikipedia.org/wiki/Простое_число), это натуральное число, имеющее ровно два различных делителя - **1** и **саму себя**.

Для нахождения начального списка простых чисел существуют следующие алгориты:
- Прямое решение
  1. определяем первое простое число = *2*
  2. для последовательности от *i = [3...N]*
  3. делим число i на предыдущие простые числа
  4. если делится с остатком, то число простое, добавляем в список простых
 
  Для оптимизаии можно пройтись по последовательности с шагом 2, т.к. все простые числы, кроме *2* являются нечётными
  Сложность алгоритма составляет ***O(n log(n))***
  
- Решето Эратосфена:
  1. выписать подряд все целые числа: *2...N*
  2. первое простое число *p* равно 2
  3. зачеркнуть в списке числа от *2p* до *N* с шагом *p*, т.е. числа, кратные *p*: *2p, 3p,...*
  4. найти первое незачёркнутое число в списке больше чем *p* и присвоить *p* это число
  5. повторить (3)-(4) если это возможно
  
  Все незачёркнутые числа в списке - простые
  Сложность алгоритма составляет ***O(n log(log(n)))***

- Решето Сандарама
  1. выписать подряд все целые числа: *2...N*
  2. исключить из ряда числа с индексом i + j + 2ij, 
     где i = *1, 2, ..., (sqrt(2N+1) - 1) / 2* и *j = i, i+1, ..., (N-i) / (2i+1)*
  3. каждое из оставшихся чисел умножить на *2* и прибавить *1*
  
  Полученная последоватьельность - простые числа в диапазоне [1, 2N + 1]
