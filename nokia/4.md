# Двоичное число  

На вход функции поступает беззнаковое целое 32битное число. 
Требуется вывести на экран максимальное и минимальное целое беззнаковое 32битное число, 
которое может быть получено с использованием такого же числа единиц в двоичном представлении, как и у оригинального числа.

Данная задача легко решается с помощью типа ***bitset*** из стандартной библиотеки.  

- Создаём 3 объекта *bitset*: 
  + **bitset<32> num(number)** - для заданного числа *number*
  + **bitset<32> min** - для формирования минимального числа
  + **bitset<32> max** - для формирования максимального числа
  
- У типа *bitset* имеется метод *count()*, который возвращает количество установленных бит.  
  Нам остаётся для **min** и **max** установить то же самое количество бит и перевести их в число.
  + для **min** - с начала
  + для **max** - с конца
